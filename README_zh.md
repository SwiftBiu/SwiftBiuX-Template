# SwiftBiuX 插件模板库 (中文)

欢迎！这里是 SwiftBiuX 官方插件模板库。

## 🚀 快速开始 (推荐)

使用我们官方的 Node.js CLI 工具，零配置瞬间生成跨端 SwiftBiu 插件脚手架：

```bash
npx create-swiftbiux-plugin <你的插件名>
```

或者，你也可以克隆此仓库来手动查看模板和源码：
```bash
git clone https://github.com/SwiftBiu/SwiftBiuX-Template.git
cd SwiftBiuX-Template
```

# 豆包图像生成插件 for SwiftBiu

欢迎使用豆包图像生成插件！这是一个轻量级的纯逻辑插件，旨在将豆包文生图模型的强大能力无缝集成到您的日常工作中。

## ✨ 功能特性

-   **后台运行**: 选中任意描述性文本作为提示词，触发插件，即可在后台调用豆包 API 生成图像。
-   **链接自动复制**: 生成的图像链接会自动复制到您的系统剪贴板，方便您在浏览器中打开或分享。
-   **系统通知**: 通过系统通知，实时了解生成任务的开始、成功或失败状态。
-   **模型与尺寸可配置**: 您可以在插件设置中，方便地选择不同的图像模型和常用的生成尺寸。

## 🚀 快速开始

在使用本插件之前，您需要从火山引擎方舟平台获取凭证并开通模型服务。

### 步骤 1: 获取 API Key

1.  访问 [火山引擎方舟控制台](https://console.volcengine.com/ark/region:ark+cn-beijing/apiKey)。
2.  使用抖音扫码登录并完成实名认证。
3.  在 API 密钥管理页面，点击“创建新的 API Key”，并妥善保管好您的 `API Key`。**插件中使用的 `API Key` 是指您的 `API Key`**。

### 步骤 2: 开通模型服务

1.  访问 [模型开通管理页面](https://console.volcengine.com/ark/region:ark+cn-beijing/openManagement)。
2.  根据您的需要，**一键开通**所有推荐的视觉模型（如 `doubao-seedream-4.5` 等）。大部分模型都提供大量的免费额度供您试用。

## 🔧 插件配置

1.  将 `DoubaoImage-Plugin` 文件夹安装到 SwiftBiu 中。
2.  打开 SwiftBiu 的“偏好设置”，找到“插件”部分。
3.  点击“豆包图像生成”插件旁边的配置按钮。
4.  **API Key**: 填入您在步骤 1 中获取的 `API Key`。
5.  **图像模型**: 从下拉菜单中选择一个您想要使用的图像模型。
6.  **图像尺寸**: 从下拉菜单中选择一个预设的生成尺寸。
7.  保存设置。

## 💡 使用方法

1.  在任何应用程序中，用鼠标选中一段描述性文字（例如：“一只戴着宇航员头盔的猫，在月球上行走”）。
2.  触发 SwiftBiu 工具栏。
3.  点击“豆包图像生成”图标。
4.  您会收到一个“正在生成中...”的通知。
5.  处理完成后，您会收到一个“图像生成成功”的通知，此时生成的**图片链接**已经复制到了您的剪贴板中，您可以直接在浏览器中粘贴并打开查看。
